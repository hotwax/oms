**Installation guide for Moqui Framework  
Command to clone Moqui-Framework repository**

    git clone https://github.com/moqui/moqui-framework.git  


**Command to change directory to moqui-framework**

    cd moqui-framework/  


**Command to clone Moqui-runtime**

    git clone https://github.com/moqui/moqui-runtime.git runtime  


**Command to change directory to runtime**

    cd runtime/



**Command to change directory to component**

    cd component/
